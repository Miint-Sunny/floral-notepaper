## 花笺 个人预览版 v1.2.0-preview.1

> [!NOTE]
> 这是维护者**个人使用的预览版**（非公开 beta）。macOS 包为 ad-hoc 签名、未公证，首次打开请右键 →「打开」。

本版集中在「即时(Live)」模式的体验打磨，并捆入此前未公开发布的几项修复（原 beta.10 草稿内容）。

### 即时模式

- **「点击弹射」根治**：切换笔记或向下滚动后，第一次点击时视口会向下「弹」一大段（越往下点弹得越远，长中文笔记尤其明显，真机量到过整篇高度）——**现在基本消除**。
  - 根因是 CodeMirror 对**屏幕外中文段落折行高度的估算**严重偏小（中文字宽是英文 2 倍却被当 1 算），首次测量时一次性释放成大跳；配合点击触发的几条滚动写入。已在编辑器内核层修准估算、并拦住点击时的多余滚动。
- **代码块块级渲染 + 语法着色**：光标不在的代码块整块渲染（行号 · 折叠 · 复制工具栏），并按语言**语法着色**（语法异步加载、就绪后自动上色；mermaid/math 不误渲染、仍显源码）。光标点进去即还原为可逐行编辑的源码。
- **活动行/块高光三档更易区分**：「行 / 块 / 块+行」原来共用同一档底色、单源码行段落下看着一样；现已拉开——**行**（更深 + 行前竖线）、**块**（整块淡染）、**块+行**（块淡染 + 当前行更深 + 竖线）。
  - 说明：「行」高亮的是整条**源码行**（一段中文长段落折行后即整段），不是 Typora 那种单条视觉行——后者与上面的点击弹射修复存在底层冲突，暂不做（设置项里也加了这条说明）。

### 笔记与便签

- **重命名笔记不再往回收站堆副本**：以前在软件内改笔记标题会"建新文件 + 把旧文件丢进回收站"，回收站越攒越多。现在改为**先写新文件、再永久删旧文件**（不进回收站），干净利落（上游 #314）。
- **修便签池隐藏窗口泄漏**：快速连开空白便签时，预热用的隐藏 WebView 可能因重叠预热/入池失败而越积越多。已加单飞守卫 + 满了即关窗，并让待机池稳定补到容量（上游 #316，Windows 影响最明显）。

### 已知小问题（不影响使用，后续再说）

- 切换/滚远后首次点击，偶尔仍有**极轻微的一帧闪动**（WKWebView 原生滚动在合成线程异步绘制，脚本层难以完全抢先）。
- 切换笔记后的**第一次点击，光标可能落在该行最前端**而非点击处；**再点同一行即准**（CodeMirror 对尚未实测的行做的是估算定位）。
