## v1.2.0-beta.7

### 设置 + 边栏/大纲 + 代码块 一大批改进 | Highlight

窄窗设置面板彻底不再压住视图切换；边栏/大纲开合与宽度现在会记忆、大纲栏可拖宽、且各有独立缩放；滚动条统一为主题绿描边并自动隐藏；新增「代码块自动换行」开关（预览/即时，默认开），预览代码块的复制/展开按钮与即时模式统一。

> [!NOTE]
> beta 预览版。macOS 包为 ad-hoc 签名、未公证，首次打开请右键 →「打开」。

#### 设置面板布局（窄窗再修）

- beta.6 那版在窄窗口下打开设置时，悬浮面板仍会压住工具栏右侧的视图切换（即时/预览）。本版让工具栏随内容增高，并按其**实测高度**定位悬浮面板，彻底消除重叠——工具栏切换始终可见可点。

#### 边栏 / 大纲

- 边栏开合、边栏宽度、大纲开合、大纲宽度现在都**记忆到配置**，重启后保留。
- **大纲栏可拖拽改宽**。
- 边栏、大纲**各有独立缩放**（设置里两个滑块，0.8–1.5×），与编辑器字号互不影响。
- 大纲表头视觉向边栏/面板设计语言看齐。

#### 滚动条

- 全应用滚动条统一成**主题绿描边滑块**；停止滚动约 3 秒后**自动隐藏**，悬停或滚动时显现。

#### 代码块

- 新增「**代码块自动换行**」开关（默认开），预览 / 分栏 / 即时一致。关闭后代码块改为横向滚动、正文照常换行。
- 预览代码块的复制按钮改为图标，并新增**折叠 / 展开**，与即时模式工具栏统一。
