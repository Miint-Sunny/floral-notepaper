## v1.2.0-beta.3

### 开窗不再白屏一闪 | Highlight

承接 beta.2 的启动提速，这一版修掉了冷启动时窗口先白屏一闪、再出现内容的问题。

> [!NOTE]
> beta 预览版。macOS 包为 ad-hoc 签名、未公证，首次打开请右键 →「打开」。

#### 等画好再开窗

主窗口原本在 webview 还没绘制时就被显示出来，透明窗口因此露出一帧白底。现在改为**等前端渲染出第一帧后才显示窗口**——窗口一出现就是有内容的状态，不再白闪。

为稳妥起见加了 3 秒兜底：万一前端没能发出"已就绪"信号，窗口也会照常显示，绝不会卡成隐形。`--silent`（开机自启到托盘）的隐藏行为不变。
