## v1.2.0-beta.5

### 启动又快回来了（且依然无白屏）| Highlight

beta.4 修好了白屏，却带来"启动奇慢"——这一版定位并修复，启动恢复迅速，白屏也不再出现。

> [!NOTE]
> beta 预览版。macOS 包为 ad-hoc 签名、未公证，首次打开请右键 →「打开」。

#### 为什么会慢，怎么修的

为了避免白屏，窗口启动时先隐藏、等前端画好首帧再显示。之前用 `requestAnimationFrame` 来判断"画好了"，但浏览器在窗口隐藏时会暂停渲染、`requestAnimationFrame` 不触发——于是显示信号迟迟发不出，窗口只能等约 3 秒的兜底超时才出现。

本版改为在 React 完成首次渲染时直接发出显示信号（不再依赖 `requestAnimationFrame`，隐藏状态下也会照常触发），窗口几乎立刻就以"有内容、无白屏"的状态出现。
