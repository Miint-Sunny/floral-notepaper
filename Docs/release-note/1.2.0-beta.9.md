## v1.2.0-beta.9

### 启动 / 卡顿 / 后台占用 优化一批 | Highlight

围绕「启动速度 / 卡顿 / 后台占用」做的一批优化：公式按需加载、后台更新检查更省、分栏预览滚动更跟手、预览动态加载更健壮。无新功能、无破坏性改动。

> [!NOTE]
> beta 预览版。macOS 包为 ad-hoc 签名、未公证，首次打开请右键 →「打开」。

#### 性能 / 后台

- **预览公式按需加载**：KaTeX（最重的预览依赖）改为**仅当笔记含公式时**才动态加载——不含公式的笔记不再为它付出加载与解析成本。
- **更新检查更省**：后台更新检查器的轮询从「每分钟空转读盘」改为**按到期时间动态休眠**；macOS 上更新器清理残留挂载点的活也**挪到后台线程**，不再挡启动 / 主线程。
- **分栏预览滚动更跟手**：编辑 ↔ 预览的滚动同步改为 **requestAnimationFrame 节流**并加固续锁，减少抖动。
- **预览更健壮**：KaTeX / Mermaid 动态加载若失败，**不再永久缓存坏结果**（后续会自动重试），并兜底未捕获的 rejection。

#### 已知问题 / 临时规避

- **即时模式：切换笔记或向下滚动后，第一次点击时视口可能向下「弹」一下**（代码块多的大文档更明显，越往下点弹得越远）。根因是 CodeMirror 对屏幕外代码块折行高度的估算误差，根治方案正在做。
  - **临时规避：设置 → 关闭「代码块自动换行」** 即可完全消除弹射（代价：代码块改为横向滚动、不再折行）。
