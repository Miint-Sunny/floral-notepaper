## v1.2.0-beta.2

### 启动更快、包更小 | Highlight

在 beta.1 全部功能的基础上，这一版专注启动提速与基础瘦身，体验不变、开得更快。

> [!NOTE]
> beta 预览版。macOS 包为 ad-hoc 签名、未公证，首次打开请右键 →「打开」。

#### 启动提速

把「即时」编辑器（CodeMirror）和「预览」渲染器（react-markdown / katex / mermaid）改为**按当前视图懒加载**——默认进入哪个视图就只加载哪一套，用不到的不再在开机时解析。小窗 / 磁贴等次要窗口也改为路由级懒加载。

- 首屏 JS 体积从约 1.5MB 降到约 0.48MB（gzip 475KB → 145KB，约 −68%）。
- mermaid 的初始化也随之移出开机路径，不画图就不会拖慢启动。

#### 基础瘦身

发布二进制启用 `strip` + thin LTO，缩小各平台安装包体积；保留默认优化级别，运行速度不受影响。
