## v1.2.0-beta.8

### 边栏排布 + 滚动条修正 + 设置窄窗 一批打磨 | Highlight

边栏顶部按钮收成一排区分图标、搜索移到其下，外部文件可拖拽重排；设置/关于窄窗时改为直接覆盖工具栏那一层、不再挤压视图切换；滚动条修正了贴边被裁（现在完整显示成胶囊）并把绿色调淡柔和。延续 beta.7 的边栏/大纲记忆与独立缩放、统一滚动条、代码块自动换行等改进。

> [!NOTE]
> beta 预览版。macOS 包为 ad-hoc 签名、未公证，首次打开请右键 →「打开」。

#### 滚动条

- **修正贴边被裁**：滑块此前贴在边栏/大纲列右缘，而列是 `overflow-hidden`，把滑块胶囊的**右侧描边裁掉了**——看着像内缩、旁边还有一道色条。现在把滚动容器收窄 2px，让滑块退进列内、**完整显示成胶囊**（两条描边 + 填充），与编辑器滚动条一致。
- 绿色滑块**调淡柔和**：不再比大纲选中项还醒目；鼠标压在滑块上时略增强作反馈。
- 全应用统一**主题绿描边滑块**，停止滚动约 3 秒后**自动隐藏**。

#### 边栏

- 顶部「新建笔记 / 导入 / 新建分类」收成计数栏那一排**区分图标**，**搜索框移到其下**，更紧凑。
- **外部文件可在边栏内拖拽重排**（会话内顺序）。
- 边栏开合、宽度记忆到配置、重启保留；可拖拽改宽；独立缩放（0.8–1.5×）。

#### 设置 / 关于面板（窄窗）

- 窄窗口下打开设置 / 关于时，面板**直接覆盖在工具栏那一层之上**，不再隐藏或挤压右侧的视图切换（即时 / 预览），也消除了隐藏闪烁。

#### 大纲

- 开合、宽度记忆；可拖拽改宽；独立缩放；表头向边栏 / 面板设计语言看齐。

#### 代码块

- 「**代码块自动换行**」开关（默认开），预览 / 分栏 / 即时一致。
- 预览代码块复制按钮改图标，新增**折叠 / 展开**，与即时模式工具栏统一。
