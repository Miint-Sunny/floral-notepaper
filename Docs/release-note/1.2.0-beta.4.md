## v1.2.0-beta.4

### 白屏闪烁真正修好了 | Highlight

beta.3 试图修白屏但没治本——这一版定位到真正的根因并修复。

> [!NOTE]
> beta 预览版。macOS 包为 ad-hoc 签名、未公证，首次打开请右键 →「打开」。

#### 根因与修复

macOS 的窗口配置（`tauri.macos.conf.json`）是对基础配置的**整体替换**，其中漏掉了"启动时先隐藏"这一项，导致窗口在前端渲染出第一帧之前就被显示出来，露出一帧白底。

本版补回该项，让 beta.3 已加入的"等首帧再显示窗口"逻辑真正生效；同时在 macOS 上让页面在加载瞬间就铺上主题底色作为兜底。两道保险叠加，冷启动不再白闪。
