## v1.2.0-beta.6

### 设置面板重排 + 即时模式两处体验修复 | Highlight

设置项按主题重新分组并加上小字说明；修复即时模式下切换笔记/外部文件被误标“未保存”、以及窄窗口下设置面板与视图切换相撞的问题。并入上游 #311（磁贴双击进编辑 / 保存后回贴）。

> [!NOTE]
> beta 预览版。macOS 包为 ad-hoc 签名、未公证，首次打开请右键 →「打开」。

#### 设置面板重排

- 一长串扁平开关 → 9 个主题分组（通用 / 外观 / 自动保存 / 编辑器 / 即时·大纲 / 磁贴与小窗 / 快捷键 / 背景 / 数据），每组带组标题。
- 关键项加上小字说明；本分支相对上游新增的项用「分支新增 / 新增」轻提示标签标出（即时·大纲组、磁贴双击/回贴）。
- 界面语言挪进「通用」、「主题」→「应用主题」、磁贴相关集中到「磁贴与小窗」、快捷记录开关文案改短。

#### 即时模式脏标记修复

- 即时(Live)模式下切换笔记/外部文件时，新载入的内容会被误判为“已修改”，导致：切走时把外部文件无谓回写磁盘（改动 mtime），或开着自动保存时光打开不编辑也会触发自动保存。
- 现已修复：仅真实编辑才标记为未保存。

#### 设置面板布局修复

- 窄窗口（< 1080px）下打开设置时，悬浮面板会压住工具栏右侧的视图切换（即时/预览）。现改为从工具栏下方展开，工具栏切换始终可见可点。

#### 其他

- 并入上游 PR #311：可选「双击磁贴进入编辑」「保存后回到磁贴」（默认关，老用户行为不变）。
- 保存按钮改用图标，与撤销/重做/删除统一。
- 补充日语(ja-JP) 即时/大纲及设置相关翻译。
