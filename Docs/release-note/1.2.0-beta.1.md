## v1.2.0-beta.1

### 一次把想要的都装进来 | Highlight

这是一个个人 beta 版：在花笺原有「编辑 / 分栏 / 预览」之外，加入了所见即所得的「即时」编辑与跨视图「大纲」，并顺手把社区里几个还没合并、但很实用的 PR 一起打了进来。多平台（Windows / Linux / macOS）一起出包。

> [!NOTE]
> 这是 beta 预览版，可能存在未发现的问题。macOS 包为 ad-hoc 签名、未公证，首次打开请右键 →「打开」。

#### 即时（Live）模式

新增第四种视图「即时」：基于 CodeMirror 6 的单栏所见即所得编辑，标题 / 强调 / 引用 / 列表 / 任务 / 表格 / 图片 / 代码块实时渲染，光标所在行显示原始 Markdown。可选当前行/块高光（关 / 行 / 块）、代码块行号与整篇行号栏、代码块折叠。

#### 大纲（Outline）

左侧大纲面板，跨「编辑 / 分栏 / 预览 / 即时」四视图：标题层级跳转、滚动/光标自动跟随当前标题（设置可关）。

#### 合并的社区改进

- mermaid 流程图在预览中渲染（upstream #293）
- 工具栏一键插入本地图片（upstream #251）、剪贴板图片检测改进（upstream #249）、图片源解析重构（upstream #247）
- 批量导入 Markdown 文件（upstream #248）
- `Ctrl + =` / `Ctrl + -` 缩放编辑器字号（upstream #287）
- 日语（ja-JP）界面（upstream #254，部分翻译，其余回退中文）

#### macOS 签名修复

发布包改为 ad-hoc 签名（`signingIdentity: "-"`），修复 Apple Silicon 上下载安装后报「已损坏，无法打开」的问题（upstream #195）。
